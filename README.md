# POWER-_BI-HOTEL-DATA-PROJECT
Business Problem
The hospitality industry faces high cancellation rates, seasonal demand fluctuations, revenue instabilityacross channels, and limited visibility into repeat guest behavior. This project provides data-driven insights to improve forecasting, pricing, and retention.

Project Overview
This project presents a comprehensive Power BI analysis of hotel booking data (2018–2020). The objective was to transform raw booking data into an analysis-ready dataset and develop an interactive dashboard supporting revenue optimization, cancellation risk management, pricing strategy, andcustomer retention.
